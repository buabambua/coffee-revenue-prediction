# coffee-revenue-prediction
# 1) การสำรวจข้อมูล (EDA)
- ข้อมูลมีทั้งหมดประมาณ 2,000 แถว
- ตัวแปรหลักประกอบด้วย
- จำนวนลูกค้าต่อวัน
- มูลค่าเฉลี่ยต่อออเดอร์
- ชั่วโมงเปิดร้านต่อวัน
- จำนวนพนักงาน
- ค่าใช้จ่ายด้านการตลาดต่อวัน
- ปริมาณคนสัญจร (Foot Traffic)
 ตัวแปรเป้าหมายคือ Daily_Revenue (รายได้ต่อวัน)

# 2) การสร้างโมเดลทำนายรายได้
ใช้ Linear Regression (เปรียบเทียบกับโมเดลที่เคยใช้ Random Forest มาก่อน) ประสิทธิภาพของโมเดล
R² Score ≈ 0.949
→ โมเดลสามารถอธิบายความแปรปรวนของรายได้ต่อวันได้ประมาณ 95% (ถือว่าดีมาก)
Mean Squared Error (MSE) ≈ 97,569.72
→ ค่าความคลาดเคลื่อนอยู่ในระดับที่ยอมรับได้เมื่อเทียบกับขนาดของรายได้

# 3) ปัจจัยที่มีผลต่อรายได้ (Coefficients)
เรียงจากมีผลเชิงบวกมาก → น้อย

- ค่า บวก (+) → เพิ่มแล้ว รายได้เพิ่ม
- ค่า ลบ (–) → เพิ่มแล้ว รายได้ลด
- ค่ายิ่ง มาก → มีผลแรงกว่า

1️ Average_Order_Value = 245.18

ความหมาย = ถ้าบิลเฉลี่ยเพิ่มขึ้น 1 USD -> รายได้ต่อวันจะเพิ่มขึ้น ≈ 245 USD

แปลเป็นภาพธุรกิจ
ถ้าขายกาแฟ + ขนม เพิ่มบิลเฉลี่ยแค่เล็กน้อย
→ รายได้ทั้งวัน “กระโดด” สูงมาก
* นี่คือปัจจัยที่แรงที่สุด

  2️ Number_of_Customers_Per_Day = 5.57

ความหมาย = ลูกค้าเพิ่ม 1 คน/วัน -> รายได้เพิ่ม ≈ 5.57 USD

แปลเป็นภาพธุรกิจ
ลูกค้าเพิ่ม 100 คน
→ รายได้เพิ่ม ≈ 557 USD (~19,500 บาท)
* จำนวนลูกค้ามีผลจริง แต่ ไม่แรงเท่าเพิ่มบิลต่อคน

  3️ Marketing_Spend_Per_Day = 1.54

ความหมาย = ลงการตลาดเพิ่ม 1 USD -> รายได้เพิ่ม ≈ 1.54 USD

แปลเป็นภาพธุรกิจ
ถ้าโฆษณาวันละ +100 USD
→ รายได้เพิ่ม ≈ 154 USD
* การตลาด “คุ้ม” ถ้าทำให้ลูกค้าเข้าจริง

  4️ Operating_Hours_Per_Day = 0.27

ความหมาย = เปิดร้านเพิ่ม 1 ชั่วโมง -> รายได้เพิ่ม ≈ 0.27 USD

* ผลน้อยมาก → เปิดนานขึ้นไม่ได้ช่วยเท่าไร
(อาจเพราะลูกค้ามาเป็นช่วงเวลาอยู่แล้ว)

5️ Location_Foot_Traffic = 0.017

ความหมาย = คนเดินผ่านเพิ่ม 1 คน -> รายได้เพิ่ม 0.017 USD

แปลเป็นภาพธุรกิจ
คนผ่านเพิ่ม 1,000 คน
→ รายได้เพิ่ม ≈ 17 USD
* ทำเลสำคัญ แต่ต้อง “ดึงให้เข้าร้าน” ด้วย

  6️ Number_of_Employees = –1.77

ความหมาย = เพิ่มพนักงาน 1 คน -> รายได้ลดลง ≈ 1.77 USD

* ไม่ได้แปลว่าพนักงานไม่ดี
แต่สะท้อนว่า ต้นทุนแรงงานสูงกว่าเงินที่สร้างเพิ่ม

# ตัวแปรที่ควรโฟกัสที่สุด

- เพิ่ม Average Order Value
- เพิ่มจำนวนลูกค้า
- ใช้การตลาดอย่างมีเป้าหมาย

# ตัวแปรที่ไม่ควรเพิ่มสุ่ม

- เปิดร้านนานขึ้น
- เพิ่มพนักงานโดยไม่เพิ่มยอดขาย

  #จากโมเดล Linear Regression พบว่า มูลค่าเฉลี่ยต่อออเดอร์เป็นปัจจัยที่ส่งผลต่อรายได้มากที่สุด รองลงมาคือจำนวนลูกค้าและงบการตลาดขณะที่จำนวนพนักงานมีผลเชิงลบต่อรายได้
